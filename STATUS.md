## 🎯 项目概况

Tyranids 是一个基于虫群智能的代码生成系统，灵感源自战锤40k的泰伦虫族。系统通过多个并行 agents 协作，利用信息素机制实现去中心化的智能涌现。

## 🐛 已知问题

### 1. Observer LLM 统计不准确 (优先级: 中)

**问题**: 报告显示 "LLM 调用: 0 次"

**原因**: `observer.recordLLMCall()` 未被调用

**影响**: 成本估算不准确（实际成本可通过 API 账单查看）

**修复计划**: Phase 11

**临时方案**: 手动计算（45 次调用 × 平均 token 数）

### 2. 代码提取偶尔失败 (优先级: 低)

**问题**: 如果 LLM 返回非标准格式，可能提取失败

**影响**: 极少见（<1% 情况）

**修复计划**: 添加多策略提取

**临时方案**: 已有 fallback 机制

### 3. 单元测试覆盖率不足 (优先级: 低)

**问题**: 部分边缘案例未测试

**影响**: 不影响主流程

**修复计划**: 持续补充
